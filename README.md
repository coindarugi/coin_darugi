3. **Inject** 버튼 클릭하여 샌드박스 환경에 주입


## 🔧 API 엔드포인트

### 1. 실시간 가격 조회
```
GET /api/prices?coins=bitcoin,ethereum,ripple
```
**응답 예시:**
```json
{
  "bitcoin": {
    "usd": 86770,
    "krw": 125620001,
    "usd_24h_change": -0.77,
    "usd_market_cap": 1732506825901
  }
}
```
**참고**: 10초 캐싱 적용으로 빠른 응답 및 API rate limit 방지

### 2. 김치 프리미엄 (개별 코인)
```
GET /api/kimchi-premium/:coinId?exchange=upbit
```
**파라미터**:
- `coinId`: 코인 ID (예: bitcoin, ethereum, ripple)
- `exchange`: 거래소 선택 (upbit, bithumb, coinone)

**응답 예시:**
```json
{
  "coinId": "bitcoin",
  "symbol": "BTC",
  "exchange": "업비트",
  "koreanPrice": 128626000,
  "globalPrice": 126097985,
  "premium": 2.0
}
```
**지원 코인**: BTC, ETH, XRP, ADA, SOL, DOT, DOGE, SHIB, MATIC, LTC, BNB, AVAX, LINK, XLM, UNI, TRX, EOS, AAVE, ALGO, ATOM

**참고**: 하위 호환성을 위해 `/api/kimchi-premium`는 비트코인으로 리다이렉트

### 3. 공포탐욕지수 (NEW)
```
GET /api/fear-greed
```
**응답 예시:**
```json
{
  "value": 24,
  "classification": "Extreme Fear",
  "timestamp": "1766534400"
}
```
**해석**:
- 0-24: 극도의 공포 (매수 기회!)
- 25-49: 공포 (조심스런 매수)
- 50: 중립 (관망)
- 51-74: 탐욕 (보유/관망)
- 75-100: 극도의 탐욕 (매도 신호!)

**참고**: 30분 캐싱 적용

## 📖 사용 설명서 (User Manual)

### 🚀 빠른 시작 가이드

#### 1️⃣ 웹사이트 접속
1. **접속**: https://crypto-darugi.com 방문
2. **메인 대시보드**: 자동으로 기본 코인들(비트코인, 이더리움, 리플)의 가격 표시

#### 2️⃣ 코인 검색 및 추가
1. **검색창 찾기**: 화면 상단의 `🔍 코인을 검색하세요...` 입력창 클릭
2. **코인 이름 입력**: 
   - 영문 이름: `Bitcoin`, `Ethereum`, `Cardano`
   - 심볼: `BTC`, `ETH`, `ADA`, `SOL`, `DOGE`
   - 한글도 가능: `비트코인`, `이더리움`
3. **검색 결과**: 자동으로 관련 코인 목록 표시 (시가총액 순위 포함)
4. **추가**: 원하는 코인 클릭 → 메인 화면에 코인 카드 추가
5. **무제한 추가**: 10,000개 이상의 전세계 모든 암호화폐 검색 및 추가 가능!

#### 3️⃣ Top 100 코인 브라우저
1. **버튼 클릭**: 검색창 옆 `Top 100 🏆` 버튼 클릭
2. **정렬 옵션 선택**:
   - **시가총액순** (기본): 가장 큰 코인부터
   - **거래량순**: 거래가 활발한 코인
   - **상승률순**: 24시간 가격 상승률 높은 순
   - **하락률순**: 24시간 가격 하락률 높은 순
3. **코인 선택**: 원하는 코인 클릭 → 메인 화면에 추가
4. **닫기**: 모달 밖 클릭 또는 `X` 버튼

#### 4️⃣ 코인 카드 이해하기
각 코인 카드에는 다음 정보가 표시됩니다:
- **코인 이름 & 심볼**: 예) Bitcoin (BTC)
- **현재 가격**: USD 및 KRW (언어 설정에 따라)
- **24시간 변동률**: 🟢 상승(초록) / 🔴 하락(빨강)
- **시가총액**: 코인의 전체 시장 가치
- **24시간 거래량**: 시장 활동성 지표
- **김치 프리미엄** (한국어만): 업비트/빗썸/코인원 3개 거래소
  - 🟢 초록 = 프리미엄 (한국이 더 비쌈)
  - 🔴 빨강 = 디스카운트 (한국이 더 싸다)

#### 5️⃣ 코인 정렬 기능
1. **정렬 버튼 위치**: 검색창 아래 `내 코인 정렬` 섹션
2. **7가지 정렬 옵션**:
   - **⭐ 즐겨찾기**: 즐겨찾기한 코인 상단 고정
   - **가격 높은순/낮은순**: 가격별 정렬
   - **변동률 높은순/낮은순**: 24시간 변동률 기준
   - **시가총액순**: 큰 코인부터 표시
   - **김프순** (한국어만): 김치 프리미엄 높은 순
   - **수익률순**: 포트폴리오 수익률 높은 순
3. **사용법**: 원하는 정렬 버튼 클릭 → 즉시 정렬 적용

#### 6️⃣ 즐겨찾기 기능
1. **추가**: 코인 카드 좌측 상단 `⭐` 별 아이콘 클릭
2. **표시**: 즐겨찾기된 코인은 **골드 테두리**로 강조
3. **정렬**: `⭐ 즐겨찾기` 정렬 옵션으로 상단 고정
4. **해제**: 별 아이콘 다시 클릭
5. **영구 저장**: 브라우저를 닫아도 즐겨찾기 유지

#### 7️⃣ 포트폴리오 관리
1. **모달 열기**: 코인 카드 하단 `포트폴리오` 버튼 클릭
2. **정보 입력**:
   - **보유 수량**: 보유한 코인 개수 (예: 0.5 BTC)
   - **평균 매수가 (USD)**: 구매한 평균 가격 (예: $50,000)
3. **실시간 계산**: 입력 시 자동으로 수익률 계산
   - 🟢 초록 = 수익 / 🔴 빨강 = 손실
4. **저장**: `저장` 버튼 클릭 → 포트폴리오에 추가
5. **포트폴리오 요약 카드**: 상단에 전체 포트폴리오 요약 표시
   - 총 투자금액
   - 현재 평가액
   - 총 수익/손실
   - 수익률 (%)

#### 8️⃣ 가격 차트 보기
1. **모달 열기**: 코인 카드 하단 `차트` 버튼 클릭
2. **기간 선택**:
   - **7일**: 최근 1주일 추세
   - **30일**: 최근 1개월 추세
   - **90일**: 최근 3개월 추세
3. **차트 인터랙션**: 마우스 호버로 정확한 가격 확인
4. **닫기**: 모달 밖 클릭 또는 `X` 버튼

#### 9️⃣ 암호화폐 뉴스 읽기
1. **뉴스 섹션**: 화면 하단 `📰 최신 암호화폐 뉴스` 섹션
2. **뉴스 보기**: 뉴스 카드 클릭 → 모달 팝업으로 전체 내용 표시
3. **번역 기능** (영어 뉴스):
   - **뉴스 카드**: `번역` 버튼 클릭 → 카드에서 바로 번역
   - **뉴스 모달**: 모달 내 `번역` 버튼 클릭 → 전체 내용 번역
   - 지원 언어: 한국어, 프랑스어, 독일어, 스페인어
   - **토글**: 번역 후 `원문` 버튼으로 영어 원문 복원
4. **더 보기**: 뉴스가 길 경우 `더 보기` 버튼으로 전체 내용 확장
5. **원문 보기**: `원문 보기` 버튼으로 CoinDesk 기사 링크 이동

#### 🔟 공포탐욕지수 활용

### 🆘 문제 해결 (Troubleshooting)

#### ❌ 가격이 로딩되지 않아요
- **원인**: CoinGecko API rate limit 또는 네트워크 문제
- **해결**: 
  1. 30초 후 자동 재시도 대기
  2. `새로고침` 버튼 클릭
  3. 페이지 새로고침 (F5 또는 Ctrl+R)

#### ❌ Top 100이 안 열려요
- **원인**: API rate limit 도달
- **해결**: 
  1. 에러 메시지 확인: "API 사용량 제한에 도달..."
  2. 검색 기능 사용 권장 (무제한)
  3. 10분 후 재시도

#### ❌ 김치 프리미엄이 안 보여요
- **원인**: 
  1. 언어가 한국어가 아님
  2. 해당 코인이 한국 거래소에 없음
- **해결**:
  1. 우측 상단에서 🇰🇷 한국어로 변경
  2. 주요 20개 코인만 김프 지원 (BTC, ETH, XRP, ADA, SOL 등)

#### ❌ 뉴스 번역이 안 돼요
- **원인**: MyMemory API 일시적 오류
- **해결**:
  1. 몇 초 후 재시도
  2. 뉴스 모달 닫았다가 다시 열기
  3. `원문 보기`로 영어 원문 확인

#### ❌ 포트폴리오가 저장 안 돼요
- **원인**: 브라우저 로컬 스토리지 비활성화
- **해결**:
  1. 브라우저 설정 → 쿠키 및 사이트 데이터 허용
  2. 시크릿 모드는 지원 안 됨

#### ❌ 모바일에서 버튼이 잘 안 눌려요
- **원인**: 터치 영역 작음
- **해결**: 버튼 중앙 부분을 정확히 탭
