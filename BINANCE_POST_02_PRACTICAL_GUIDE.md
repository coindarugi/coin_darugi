## 🤔 왜 여러 거래소 가격을 비교해야 할까?

많은 분들이 한 거래소만 보고 매매하시는데요, **큰 손해**를 보실 수 있습니다!

### 실제 사례:
```
비트코인 가격 (2024년 12월 기준)
🇺🇸 바이낸스 US:    $97,500
🇰🇷 업비트:         ₩142,000,000 ($98,200)
🇰🇷 빗썸:           ₩142,500,000 ($98,500)

→ 같은 시간, 최대 $1,000 차이! 💰
```

**이게 바로 "김치 프리미엄"입니다.**

한국 거래소가 해외보다 비싸면:
- ✅ 해외에서 사서 → 한국에서 팔기 (차익거래)
- ⚠️ 한국에서 비싸게 사지 않기

## 📊 시가총액 vs 거래량, 뭘 봐야 할까?

### 1️⃣ **시가총액 (Market Cap)**
```
시가총액 = 현재가 × 총 발행량
```

**의미:**
- 코인의 "몸값" = 시장에서 얼마나 큰 코인인가?
- Top 100 안에 들면 "메이저 코인"

**활용법:**
- 🟢 시총 상위권: 안정적 투자 (BTC, ETH)
- 🟡 시총 중위권: 성장 가능성 (신규 알트코인)
- 🔴 시총 하위권: 고위험 고수익

---

### 2️⃣ **거래량 (Volume)**
```
24시간 거래량 = 하루 동안 거래된 총 금액
```

**의미:**
- "인기도" = 사람들이 얼마나 활발히 사고파는가?
- 거래량 높으면 → 유동성 좋음 → 원하는 가격에 매매 쉬움

**활용법:**
- ✅ 거래량 급증 = 관심 폭발 (호재 발생?)
- ⚠️ 거래량 감소 = 관심 하락 (조정 가능성?)
- 🚨 거래량 없는데 가격 급등 = 펌프앤덤프 의심

---

### 3️⃣ **등락률 (24h Change)**
```
24시간 등락률 = (현재가 - 전일가) / 전일가 × 100
```

**의미:**
- 하루 동안 얼마나 올랐는가/떨어졌는가?

**활용법:**
- 🟢 +10% 이상: 강한 상승 (단, 과열 주의)
- 🔴 -10% 이하: 급락 (공포 매도 or 저점 매수 기회?)
- ⚪ ±3% 이내: 안정적 움직임

## 🛠️ 무료 도구로 프로처럼 분석하는 법

### 📌 **Step 1: 관심 코인 찾기**

**Top 100 브라우저 활용:**
1. 시가총액순 정렬 → 안정적 코인 찾기
2. 거래량순 정렬 → 핫한 코인 찾기  
3. 등락률순 정렬 → 급등/급락 코인 찾기

**팁:** 3개 지표를 교차 분석!
```
예: 거래량 Top 10 + 등락률 +5% 이상
→ 지금 시장에서 가장 뜨거운 코인!
```

---

### 📌 **Step 2: 가격 비교 & 김치 프리미엄 확인**

**3개 한국 거래소 실시간 비교:**
- 업비트, 빗썸, 코인원 동시 체크
- 김치 프리미엄 % 자동 계산

**활용 시나리오:**
```
상황 1: 김치 프리미엄 +3%
→ 한국이 비싸다 = 해외 거래소 체크

상황 2: 김치 프리미엄 -2%  
→ 한국이 싸다 = 매수 기회?

상황 3: 김치 프리미엄 0%
→ 정상 범위 = 평소처럼 거래
```

---

### 📌 **Step 3: AI 전망 분석 활용**

**AI가 분석해주는 것들:**
- 📈 최근 가격 추세 (상승/하락/횡보)
- 📊 거래량 변화 패턴
- 🔮 단기 전망 (주의: 참고용!)

**올바른 사용법:**
```
❌ 나쁜 예: AI가 상승 예측했으니 올인!
✅ 좋은 예: AI 분석 + 내 판단 + 리스크 관리
```

**AI는 도구일 뿐, 최종 결정은 본인이!**
