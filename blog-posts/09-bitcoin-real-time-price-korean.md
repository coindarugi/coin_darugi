## 왜 실시간 시세 추적이 중요한가?

### 1. 매매 타이밍 포착
```
예시: 2024년 12월
- 09:00 - BTC ₩88,500,000
- 09:30 - BTC ₩92,000,000 (+4%)
- 10:00 - BTC ₩89,200,000 (-3%)

30분 안에 +4% → -3% 변동
→ 실시간 추적으로 매매 기회 포착
```

### 2. 손실 방지
```
설정한 손절가: ₩85,000,000
실시간 가격: ₩84,800,000
→ 즉시 매도하여 추가 손실 방지
```

### 3. 차익거래 기회
```
업비트: ₩89,500,000
빗썸: ₩89,200,000
차이: ₩300,000 (0.34%)
→ 실시간 비교로 차익거래 가능
```

## 실시간 시세 활용 전략

### 전략 1: 단타 매매 (분/시간 단위)

**필요 조건**:
- 30초 이내 업데이트
- 빠른 주문 체결
- 낮은 수수료 (0.05-0.1%)

**수익 목표**: 거래당 0.5-2%
**일일 거래**: 3-10회
**위험도**: 높음

**예시**:
```
09:00 - ₩88,000,000 매수
09:15 - ₩88,700,000 매도
수익: ₩700,000 (0.8%)
투자금: ₩10,000,000
실수익: ₩70,000 (수수료 제외)
```

### 전략 2: 스윙 트레이딩 (일/주 단위)

**필요 조건**:
- 1-5분 업데이트
- 기술적 분석
- 뉴스 모니터링

**수익 목표**: 거래당 5-15%
**월간 거래**: 2-8회
**위험도**: 중간

**예시**:
```
월요일 - ₩85,000,000 매수
금요일 - ₩95,000,000 매도
수익: ₩10,000,000 (11.8%)
```

### 전략 3: 김치 프리미엄 차익거래

**필요 조건**:
- 한국/해외 거래소 실시간 비교
- 프리미엄 5% 이상 시 실행
- 빠른 송금 가능

**수익 목표**: 거래당 3-8%
**기회 빈도**: 월 1-3회
**위험도**: 중간

**예시**:
```
코인베이스: $65,000 (₩86,500,000)
업비트: ₩92,000,000
프리미엄: 6.4%

1. 코인베이스에서 1 BTC 매수
2. 업비트로 전송 (30분-2시간)
3. 업비트에서 매도
수익: ₩5,500,000 (송금비/수수료 제외)
```

## 기술적 지표 활용법

### 1. 이동평균선 (Moving Average)

```
20일선: 단기 추세
- 가격이 20일선 위: 상승 추세
- 가격이 20일선 아래: 하락 추세

50일선: 중기 추세
200일선: 장기 추세

골든크로스: 50일선이 200일선 상향 돌파 (강력한 매수 신호)
데드크로스: 50일선이 200일선 하향 돌파 (강력한 매도 신호)
```

### 2. RSI (상대강도지수)

```
RSI 수치: 0-100
- 70 이상: 과매수 (매도 고려)
- 30 이하: 과매도 (매수 고려)
- 40-60: 중립 구간
```

### 3. 거래량 분석

```
가격 상승 + 거래량 증가 = 강한 상승 (신뢰도 높음)
가격 상승 + 거래량 감소 = 약한 상승 (주의 필요)
가격 하락 + 거래량 증가 = 강한 하락 (위험)
가격 하락 + 거래량 감소 = 약한 하락 (반등 가능)
```

## 자주 하는 실수

### 1. 과도한 모니터링

```
문제: 1분마다 가격 확인
→ 스트레스 증가, 불필요한 매매

해결: 정해진 시간에만 확인 (예: 하루 3회)
```

### 2. 감정적 매매

```
문제: -5% 하락 시 패닉 매도
→ 손실 확정, 반등 기회 놓침

해결: 미리 손절가 설정, 계획대로 실행
```

### 3. 수수료 무시

```
문제: 하루 10회 매매
수수료: 10회 × 0.1% × 2 = 2%

해결: 거래 횟수 줄이기, 수익 > 수수료 확인
```

### 4. FOMO (Fear of Missing Out)

```
문제: +50% 급등 후 매수
→ 고점 매수, 조정 시 큰 손실

해결: 기다림의 미학, 조정 후 매수
```
